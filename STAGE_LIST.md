更新日: 2026-05-13（v3）

## 凡例

- **バリアント**: 同コードでクリアが必要なマップ数
- **クリア条件**: `goal`=ゴール到達 / `goal+items`=全アイテム取得後ゴール / `put`=敵へPUT / `survive`=N ターン生存 / `items`=N 個取得
- **ブロックセット**: BASIC → CHECK → STATE → LOOK → SEARCH → ENEMY → COUNT
- **状態**: ✅ 再設計済 / ⚙️ 既存実装を移動 / 🆕 新規設計
